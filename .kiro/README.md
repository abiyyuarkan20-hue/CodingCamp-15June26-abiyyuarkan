## About the Project

**Life Dashboard To-Do List** is a professional, feature-rich productivity dashboard built with vanilla HTML, CSS, and JavaScript.

### Key Features

- ⏰ **Live Clock & Date** - Real-time display with time-based greetings
- 👤 **Custom Name Greeting** - Personalize your dashboard with your name
- 🎯 **Focus Timer** - 25-minute Pomodoro timer with start/stop/reset controls
- ✅ **To-Do List** - Create, edit, mark complete, and delete tasks
- 🔗 **Quick Links** - Save your favorite websites for fast access
- 🌙 **Dark/Light Mode** - Professional theme toggle

### Challenges Implemented

1. **Light/Dark Mode** - Toggle between professional light and dark themes
2. **Custom Name in Greeting** - Personalize the dashboard with your name
3. **Prevent Duplicate Tasks** - Ensure all tasks are unique
4. **Sort Tasks** - Sort by default, alphabetically (A-Z), or by completion status

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables and gradients
- **Vanilla JavaScript** - No frameworks, pure ES6+

## Local Storage

All user data is stored locally using the browser's Local Storage API:
- User name
- Tasks list
- Quick links
- Theme preference

## Getting Started
