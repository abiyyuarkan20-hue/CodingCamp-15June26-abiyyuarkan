## 📞 Support & Help

### Common Issues

**Website not loading?**
- Wait 1-2 minutes for initial deployment
- Clear browser cache (Ctrl+Shift+Del)
- Try different browser
- Check internet connection

**Data not persisting?**
- Check if browser allows Local Storage
- Ensure not in Private/Incognito mode
- Check Local Storage limit (usually 5-10MB)

**Theme not saving?**
- Clear browser cache
- Check Local Storage in DevTools
- Disable browser extensions that block storage

### Debug with DevTools

Press `F12` to open Developer Tools:

1. **Console Tab:** View any errors
2. **Application Tab:** Check Local Storage
   - Expand "Local Storage"
   - Select the domain
   - View stored data

3. **Network Tab:** Check resource loading
